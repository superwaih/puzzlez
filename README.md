# Public Puzzle Game (Next.js + TypeScript)

A standalone sliding puzzle game app ready to deploy publicly.

## Run locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Build

```bash
npm run build
npm start
```

## Deploy

Deploy quickly on [Vercel](https://vercel.com/new).

## Notes

- Game route: `/games`
- Puzzle image path: `public/valentine-puzzle.jpg`
